Hello, world!

This is a template for running a static site inside of a iOS App. You can use this to test out a design of a site or how it performs in iOS app. JavaScript inside of iOS can be a big flaky, so it's good to test that out in something like this. This could also be great for usability testing, especially if you don't want to rely on a remote server and be able to bundle everything inside the www folder of this app.
